# Tatva

A collection of framework-agnostic Web Components designed for composability, accessibility, and ease of use.

## Overview

Tatva provides a set of reusable Custom Elements you can drop into any project. Components are self-contained, style-safe via Shadow DOM, and expose a clean attribute-based API.

### Features

- Works with any framework (React, Vue, Svelte, Angular, or plain JS)
- Encapsulated, conflict-free styles with Shadow DOM
- Accessible defaults and keyboard support
- Simple, attribute-based APIs
- Slot support for inline content

### Technical

- Components live in: `src/elements/`
- Register once: `src/elements/index.js`
- Configure with HTML `attributes`
- Some components support `slots` for inline content

## Installation

### via CDN

```html
<script type="module" src="https://unpkg.com/@lubusin/tatva/dist/index.js"></script>
```

### via NPM

```bash
npm install @lubusin/tatva
```

## Usage

### Setup

#### Load all components:

```js
// main.js
import '@lubusin/tatva/index.js';
```

#### Load a single component:

```js
import '@lubusin/tatva/tatva-marker.js';
```

### Examples

#### Marker

```html
<tatva-marker type="underline" color="#dc143c" animation animation-duration="5s">
  Highlighted text
</tatva-marker>
```

#### Pulse

```html
<tatva-pulse position="superscript" size="0.75rem" color="#1e90ff" gap="8px" superscript-offset="-0.5em">
  Pro
</tatva-pulse>
```

Explore Storybook to know more about various eleements and customization avaiable

## **Tip**

In frameworks like React, prefer attribute strings over camelCase props for Custom Elements, and listen to native DOM events if exposed.
