# Tatva Marker

<Subtitle>Draws an animated SVG marker (underline, curly, strike, etc.) around slotted text.</Subtitle>

<Description>
The `tatva-marker` component wraps its slotted content and renders an SVG path above or below the text to create a marker-like effect. It supports multiple styles, color customization, and optional animation.
</Description>

## Usage

```html
<tatva-marker type="underline" color="#dc143c" animation animation-duration="5s" animation-function="ease-in">
  Highlighted content
</tatva-marker>
```

## Props / Attributes

<ArgsTable of={import('./tatva-marker.stories.js')} />

- `type`: Visual style of the marker. Options: `circle`, `curly`, `underline`, `double`, `double-underline`, `underline-zigzag`, `strikethrough`, `cross`, `strike`.
- `color`: Stroke color of the marker path (CSS color string).
- `animation`: When present, enables marker drawing animation.
- `animation-duration`: Duration of the animation (e.g., `5s`).
- `animation-function`: CSS timing function (e.g., `ease-in`).

## Best Practices

- Choose a `type` that complements the text context (e.g., `underline` for emphasis, `strikethrough` for negation).
- Keep `animation` subtle; fast or infinite animations can be distracting. Consider disabling for long paragraphs.
- Ensure color contrast between the marker and background for readability.
